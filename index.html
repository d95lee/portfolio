Below are some of the tools and frameworks I've used for my projects

Amazon Clone

A clone of Amazon.com created using a ruby on rails backend with a react-redux frontend. Products are categorized and with the simple click of a button, you can view the product name, price, and description as well as view the ratings of each product on the product show page. Each product can have many reviews that can be written by the user and viewed by any user whether or not they're logged in.

Compass Travel Itinerary

Compass is a travel planning application that utilizes a mongoose, express backend that serves the react-redux, node.js frontend created by a group of 4 software engineers. Users can create a profile and create a travel itinerary that consists of 4 major parts. The first being the travel itinerary information which will also encapsulate event, transportation, and housing information.

Zombie-run

Zombie-run is a zombie shooting game created using JavaScript. The goal of Zombie Run is to survive the apocalypse where zombies have taken over the courtyard. The objective is to survive for as many levels as possible in either easy or hard mode as the number of zombies increase. Users can move using the ‘wasd’ keys while clicking on the zombies to eliminate them.
